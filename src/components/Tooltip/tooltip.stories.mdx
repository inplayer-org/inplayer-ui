# Tooltip

Tooltip a is a component for displaying additional information on hover some element.
The position of the tooltip attribute can be: `up`, `down`, `right`, `left`.

```jsx
import { Tooltip } from '@inplayer-org/inplayer-ui';

const Page = () => (
  <Tooltip />
)
```
