# Header

The Header represents a versatile component which can house a title or a TabNavigation, 
as well as a UserMenu and Additional User Menu.

```jsx
import { Header } from '@inplayer-org/inplayer-ui';

const Page = () => (
  <Header />
)
```

<Props of={Header}/>
