# Loader

A simple spinner/loader to use as fallback when doing async operations. By default it inherits the `primary.main` theme color. 

Can be passed a `color` prop with a custom hex color.

```jsx
import { Loader } from '@inplayer-org/inplayer-ui';

const Page = () => (
  <Loader />
)
```
