# Navbar

The Navbar is a horizontal navigation bar component. It takes an image element as the 
`logo` icon at the top. Menu items are rendered as children of the Navbar using the Navbar.MenuItem component. 
The MenuItem takes an `icon` as a prop (as a React Node), and an `active` prop, 
which indicates whether the selected menu item is active. `onClick` is an optional prop which lets you add 
custom click logic to the MenuItem. The MenuItem can in an `a` or `Link` from react-router element as well, 
just make sure to clear the text-decoration.

```jsx
import { Navbar } from '@inplayer-org/inplayer-ui';

const Page = () => (
  <Navbar />
)
```
